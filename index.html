Scroll Snap Slider

Mostly CSS Slider with great performance

These demos include some additional UI. The Arrows and dots are not included in the library, but code samples can be copied from the demo code.

Premise

This library is a shortcut to something I personally have to implement in almost every website. To keep it small, there are not many fancy features, no error handling.

However, with a clear API and the use of a ES6 class, it can provide a useful base for custom extensions.

What this module contains:

  • Example markup for a scroll-snap slider
  • CSS default styling for a scroll-snap slider without scrollbars
  • ES6 class to slightly enhance functionality
  • ES6 class plugins for loop , autoplay , and draggable features

For a more "fully-featured" implementation, go to Tanner Hodges' snap-slider
